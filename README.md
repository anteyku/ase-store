# Верстка магазин CS-GO кейсов
![Header](headercsgo.jpg)

Чистая верстка магазина cs-go кейсов. Дизайн сделан нами. Макет сверстан за неделю. При его создании все картинки предметов в кейсах были взяты из открытого доступа в интернете. Сайт предназначался для открытия кейсов cs-go. Адаптивен под все телефонные устройства. Хотим уточнить что это только верстка и она не имеет логики или бекенда.
